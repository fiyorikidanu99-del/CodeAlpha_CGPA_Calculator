# CGPA Calculator

## Project Description

A C++ console-based application that calculates a student's CGPA based on course information, credit hours, and grades.

This project was developed as part of the CodeAlpha C++ Programming Internship.

## Features

- User can enter multiple courses
- Stores course name, credit hours, and grades
- Converts grades into grade points
- Calculates total credit hours
- Calculates CGPA automatically
- Displays a formatted course summary table

## Technologies Used

- C++
- Standard Template Library (STL)
- Object-oriented programming concepts
- File organization and debugging practices
